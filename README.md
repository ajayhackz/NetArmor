## Overview

NetAromr is a simple Python script designed to scan and analyze outgoing traffic, providing detailed information about each IP address. It doesn't just stop there - it's equipped with a sophisticated mechanism to detect potential malicious activity. If a server's IP address is flagged as malicious using the VirusTotal API, Netguard ensures you're in the know! üîç

## Features
 
    Traffic Scanning: Get detailed insights into all outgoing traffic from your Windows computer.
    Malicious Activity Detection: Utilize the power of the VirusTotal API to identify and flag potentially malicious server IP addresses.


## How to Use

1. Clone the repository to your local machine.

        git clone https://github.com/viralvaghela/netguard.git

2. Navigate to the project directory.

        cd netguard

3. Run the script.
